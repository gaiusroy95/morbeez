# Mobile apps moved

Staff and farmer mobile clients now live under **`apps/`**:

- `apps/farmer` — farmer shopping + workspace
- `apps/warehouse` — pick & pack
- `apps/agronomist` — agronomist visits, farmer intelligence, and tasks

See [docs/MOBILE-APPS.md](../docs/MOBILE-APPS.md).

```bash
npm run dev:farmer
npm run dev:warehouse
npm run dev:agronomist
```
